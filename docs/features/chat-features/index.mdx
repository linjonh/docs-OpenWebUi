# 聊天功能概览

Open WebUI 提供了一套全面的聊天功能，旨在增强您与 AI 模型的交互体验。本页面概述了主要聊天功能，并提供链接以查阅更详细的信息。

## 核心聊天功能

- **[🗂️ 会话组织](./conversation-organization.md)**：使用文件夹和标签整理聊天，使工作空间保持整洁和有条理。

- **[🔗 URL 参数](./url-params.md)**：通过 URL 参数配置聊天会话，快速设置模型、工具及其他功能。

- **[⚙️ 聊天参数](./chat-params.md)**：在不同层级（每个聊天、每个账户或每个模型）控制系统提示和高级参数。

- **[🗨️ 聊天分享](./chatshare.md)**：通过本地或 Open WebUI 社区平台分享会话，并可控制隐私设置。
