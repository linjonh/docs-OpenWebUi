# 채팅 기능 개요

Open WebUI는 AI 모델과의 상호작용을 강화하기 위해 설계된 포괄적인 채팅 기능 세트를 제공합니다. 이 페이지는 주요 채팅 기능에 대한 개요를 제공하며, 자세한 정보를 위한 전용 페이지 링크를 포함하고 있습니다.

## 핵심 채팅 기능

- **[🗂️ 대화 정리](./conversation-organization.md)**: 파일 및 태그를 사용하여 채팅을 정리하고 작업 공간을 깔끔하고 체계적으로 유지하세요.

- **[🔗 URL 매개변수](./url-params.md)**: URL 매개변수를 통해 채팅 세션을 구성하여 모델, 도구, 기타 기능을 빠르게 설정할 수 있습니다.

- **[⚙️ 채팅 매개변수](./chat-params.md)**: 시스템 프롬프트 및 고급 매개변수를 다양한 수준(채팅별, 계정별, 모델별)에서 제어하세요.

- **[🗨️ 채팅 공유](./chatshare.md)**: 로컬이나 Open WebUI 커뮤니티 플랫폼을 통해 대화를 공유하고 프라이버시 설정을 제어할 수 있습니다.
