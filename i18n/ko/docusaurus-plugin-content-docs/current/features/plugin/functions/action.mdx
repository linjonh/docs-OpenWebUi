액션 함수는 최종 사용자가 상호작용할 수 있도록 메시지 도구 모음에 사용자 정의 버튼을 작성할 수 있도록 합니다.
이 기능은 더 상호작용적인 메시징을 가능하게 하며, 사용자가 작업이 실행되기 전에 허락을 부여하거나, 구조화된 데이터의 시각화를 생성하거나, 채팅 오디오 클립을 다운로드하거나,
다양한 기타 사용 사례를 지원합니다.

액션 코드의 템플릿은 [커뮤니티 섹션](https://openwebui.com/f/hub/custom_action/)에서 확인할 수 있습니다.

그래프 시각화 액션의 예는 아래 비디오에서 확인할 수 있습니다.

### Action
액션은 메시지 UI(개별 채팅 메시지 바로 아래에 표시되는 작은 버튼)에 버튼을 생성하는 데 사용됩니다.

액션은 액션 함수라는 단일 주요 구성 요소가 있습니다. 이 구성 요소는 액션의 타입과 처리되는 데이터를 정의하는 객체를 받습니다.

<details>
<summary>예제</summary>

```python
async def action(
        self,
        body: dict,
        __user__=None,
        __event_emitter__=None,
        __event_call__=None,
    ) -> Optional[dict]:
        print(f"action:{__name__}")

        response = await __event_call__(
            {
                "type": "input",
                "data": {
                    "title": "메시지 작성하기",
                    "message": "여기에 추가할 메시지를 작성하세요",
                    "placeholder": "메시지를 입력하세요",
                },
            }
        )
        print(response)
```
</details>
