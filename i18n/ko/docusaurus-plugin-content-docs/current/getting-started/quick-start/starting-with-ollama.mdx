## 개요

Open WebUI는 **Ollama** 인스턴스를 연결하고 관리하기 쉽게 만들어줍니다. 이 가이드에서는 연결 설정, 모델 관리 및 시작하는 방법에 대해 안내합니다.

---

## 단계 1: Ollama 연결 설정

Open WebUI가 설치되고 실행되면 자동으로 Ollama 인스턴스에 연결을 시도합니다. 모든 것이 순조롭게 진행되면 바로 모델을 관리하고 사용할 준비가 됩니다.

하지만 연결 문제에 직면하는 경우, 가장 일반적인 원인은 네트워크 구성 오류입니다. 이러한 문제를 해결하려면 [연결 문제 해결 가이드](/troubleshooting/connection-error)를 참조하십시오.

---

## 단계 2: Ollama 인스턴스 관리

Open WebUI에서 Ollama 인스턴스를 관리하려면 다음 단계를 따르십시오:

1. Open WebUI에서 **어드민 설정**으로 이동합니다.
2. **연결 > Ollama > 관리**로 이동합니다 (스패너 아이콘 클릭).  
   여기에서 모델을 다운로드하고 설정을 구성하며 Ollama와의 연결을 관리할 수 있습니다.

관리 화면은 다음과 같습니다:

![Ollama 관리 화면](/images/getting-started/quick-start/manage-ollama.png)

![Ollama 관리 화면](/images/getting-started/quick-start/manage-modal-ollama.png)


## 모델을 빠르고 효율적으로 다운로드하는 방법

빠르게 시작하려는 경우 **모델 선택기**에서 직접 모델을 다운로드할 수 있습니다. 원하는 모델 이름을 입력하면, Open WebUI가 해당 모델을 다운로드할 수 있도록 Ollama에서 요청할 것입니다.

작동 방식 예시는 다음과 같습니다:

![Ollama 다운로드 요청](/images/getting-started/quick-start/selector-ollama.png)

이 방법은 어드민 설정 메뉴를 탐색하지 않고 바로 모델 사용으로 넘어가고 싶을 때 완벽합니다.

---

## 준비 완료!

이게 전부입니다! 연결이 구성되고 모델이 다운로드되면, Ollama를 Open WebUI와 함께 사용할 준비가 됩니다. 새로운 모델을 탐색하거나 기존 모델을 실행하든, Open WebUI는 모든 것을 간단하고 효율적으로 만들어줍니다.

문제가 발생하거나 추가 지침이 필요하면 [도움말 섹션](/troubleshooting)에서 자세한 해결 방법을 확인하십시오. Ollama를 즐겨 사용하세요! 🎉
