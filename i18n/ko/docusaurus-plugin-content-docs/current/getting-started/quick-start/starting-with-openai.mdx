## 개요

Open WebUI는 OpenAI 및 기타 OpenAI 호환 API를 연결하여 사용하는 과정을 간소화합니다. 이 가이드에서는 API 키 추가, 올바른 엔드포인트 설정, 모델 선택 방법을 안내하여 바로 채팅을 시작할 수 있도록 도와드립니다.

---

## 1단계: OpenAI API 키 가져오기

OpenAI 모델(GPT-4 또는 o3-mini 등)을 사용하려면 지원 제공자로부터 API 키가 필요합니다.

다음을 사용할 수 있습니다:

- OpenAI 직접 (https://platform.openai.com/account/api-keys)
- Azure OpenAI
- OpenAI 호환 서비스(예: LocalAI, FastChat, Helicone, LiteLLM, OpenRouter 등)

👉 키를 얻은 후 복사하여 보관하세요.

대부분의 OpenAI 사용을 위해 기본 API URL은 다음과 같습니다:  
https://api.openai.com/v1

다른 제공자는 다양한 URL을 사용하니 제공자의 문서를 확인하세요.

---

## 2단계: Open WebUI에 API 연결 추가

Open WebUI를 실행한 후:

1. ⚙️ **관리 설정(Admin Settings)**으로 이동합니다.
2. **연결 > OpenAI > 관리(Manage)**(스패너 아이콘을 찾으세요)로 이동합니다.
3. ➕ **새 연결 추가(Add New Connection)**를 클릭합니다.
4. 다음을 입력합니다:
   - API URL: https://api.openai.com/v1 (또는 제공자의 특정 URL)
   - API 키: 여기에 키를 붙여넣으세요

5. 저장✅을 클릭합니다.

이렇게 하면 당신의 자격 증명이 안전하게 저장되고 연결이 설정됩니다.

다음과 같은 모습입니다:

![OpenAI 연결 화면](/images/getting-started/quick-start/manage-openai.png)

## 3단계: 모델 사용 시작

연결이 저장되면 Open WebUI 내에서 바로 모델을 사용할 수 있습니다.

🧠 모델을 다운로드할 필요가 없습니다 — 단순히 모델 선택기(Model Selector)에서 하나를 선택하고 채팅을 시작하세요. 제공자가 지원하는 모델이라면 그들의 API를 통해 즉시 사용할 수 있습니다.

모델 선택 모습은 다음과 같습니다:

![OpenAI 모델 선택기](/images/getting-started/quick-start/selector-openai.png)

제공자가 제공하는 GPT-4, o3-mini 또는 호환 모델을 간단히 선택하세요.

## 준비 완료!

이제 OpenAI 호환 API 연결을 사용할 준비가 되었습니다.

Open WebUI와 OpenAI를 통해 강력한 언어 모델과 직관적인 인터페이스, 빠른 채팅 기능을 이용할 수 있습니다 — 번거로운 설정 없이 바로 사용할 수 있습니다.

문제가 발생하거나 추가 지원이 필요하면 [도움말 섹션](/troubleshooting)을 방문하세요.

행복한 프롬프트 작업 되세요! 🎉
