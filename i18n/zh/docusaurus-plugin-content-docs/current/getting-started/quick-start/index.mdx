:::info **关于用户角色和隐私的重要提示：**

- **管理员创建：** 在 Open WebUI 上创建的第一个账号将自动获得**管理员权限**，可以管理用户和系统设置。
- **用户注册：** 随后的注册账号起初为**待处理**状态，需经管理员批准才能访问。
- **隐私和数据安全：** **您的所有数据**，包括登录信息，都是**本地存储**在设备上。 Open WebUI 确保**严格保密**，且**无外部请求**以增强隐私和安全性。
  - **所有模型默认私密。** 模型需要显式通过群组或公开设置共享。如果模型分配给某个群组，只有该群组成员可以查看。如果模型设置为公开，实例上的任何人都可以查看。

:::

选择您偏好的安装方式：

- **Docker：** **官方支持，推荐大多数用户使用**
- **Python：** 适用于资源有限的环境或希望进行手动设置的用户
- **Kubernetes：** 适合需要扩展和编排的企业部署
