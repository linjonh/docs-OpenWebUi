# 聊天功能概览

Open WebUI 提供了一套全面的聊天功能，旨在提升您与 AI 模型的互动体验。本页面概述了主要的聊天功能，并提供指向专门页面的链接以获取更详细的信息。

## 核心聊天功能

- **[🗂️ 对话组织](./conversation-organization.md)**：通过文件夹和标签组织聊天，保持工作空间整洁有序。

- **[🔗 URL 参数](./url-params.md)**：通过 URL 参数配置聊天会话，快速设置模型、工具和其他功能。

- **[⚙️ 聊天参数](./chat-params.md)**：在不同级别（每个聊天、每个账户或每个模型）控制系统提示和高级参数。

- **[🗨️ 聊天共享](./chatshare.md)**：通过本地或 Open WebUI 社区平台共享对话，并可控设置隐私选项。
