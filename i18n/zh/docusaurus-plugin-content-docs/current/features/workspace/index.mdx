在 Open WebUI 的工作区中，提供了一个全面的环境来管理您的人工智能交互和配置。它由多个关键组件组成：

## 资源

- [🤖 模型](./models.md) - 创建和管理针对特定目的定制的模型
- [🧠 知识](./knowledge.md) - 管理您的知识库以增强生成效果
- [📚 提示](./prompts.md) - 创建并组织可重复使用的提示

工作区的每个部分都旨在让您能够对 Open WebUI 的体验进行细粒度的控制，从而实现人工智能交互的定制和优化。

## 访问控制
- [‍🔑 角色](./roles.md) - Open WebUI 中定义的角色
- [‍🔐 群组](./groups.md) - 设置用户群组以共享资源访问权限
- [🔒 权限](./permissions.md) - 配置访问控制和功能可用性

角色、群组和权限设计为协同工作，以提供一个细粒度的模型，用于控制用户对单个资源的访问。
