# ⚙️ 什么是工具？

工具是一些小型的 Python 脚本，可为您的 LLM 增强功能。当启用工具时，它们可以使您的聊天机器人完成非凡的任务，比如搜索网页、抓取数据、生成图像、使用 AI 语音进行回复等等。

将工具视为您的 AI 在与您聊天时可以使用的有用插件。

---

## 🚀 工具可以帮助我完成什么？

以下是工具可以让您的 AI 助手完成的一些示例：

- 🌍 网页搜索：通过搜索互联网获取实时答案。
- 🖼️ 图像生成：根据您的提示创建图像。
- 🔊 语音输出：使用 ElevenLabs 生成 AI 语音。

在这里探索现成的工具：  
🧰 [工具展示](https://openwebui.com/tools)

---


## 📦 如何安装工具

在 Open WebUI 中安装工具有两种简单方法：

1. 访问 [社区工具库](https://openwebui.com/tools)
2. 选择一个工具，然后点击“获取”按钮。
3. 输入您的 Open WebUI 实例的 IP 地址或 URL。
4. 点击“导入到 WebUI”——完成！

🛑 安全提示：永远不要导入您不认识或不信任的工具。这些是 Python 脚本，可能会运行不安全的代码。

---


## 🔧 如何在 Open WebUI 中使用工具

一旦您安装了工具（我们稍后会告诉您如何安装），以下是启用和使用工具的方法：

您有两种方式为模型启用工具：

### ➕ 选项 1：从聊天窗口启用

聊天时，点击输入区域中的 ➕ 图标。您会看到可用的工具列表——您可以即时为会话启用任何一个工具。

💡 提示：启用工具给予模型使用工具的权限——但它可能仅在任务需要时使用工具。

### ✏️ 选项 2：默认启用（推荐频繁使用）
1. 进入：工作区 ➡️ 模型
2. 选择您正在使用的模型（如 GPT-4 或 LLaMa2），然后点击 ✏️ 编辑图标。
3. 滚动到“工具”部分。
4. ✅ 勾选您想要默认启用的工具。
5. 点击保存。

这样可以确保模型在您与其聊天时始终准备好使用这些工具。

您还可以使用 AutoTool Filter 让您的 LLM 自动选择合适的工具：

🔗 [AutoTool Filter](https://openwebui.com/f/hub/autotool_filter/)

🎯 注意：即使使用 AutoTool，您仍需通过选项 2 启用工具。

✅ 就这样 —— 您的 LLM 现在具备工具功能了！您已准备好通过网页搜索、图像生成、语音输出等功能增强聊天体验。

## 🧠 选择工具的使用方式：默认 vs 原生

工具启用后，Open WebUI 为您的 LLM 提供两种不同的方式在对话中使用它们。

您可以通过选择以下模式决定模型如何调用工具：

- 🟡 默认模式（基于提示）
- 🟢 原生模式（内置函数调用）

让我们详细看看：

### 🟡 默认模式（基于提示的工具触发）

这是 Open WebUI 的默认设置。

在此模式下，您的 LLM 不需要本地支持函数调用。相反，我们使用智能工具选择提示模板来指导模型选择和使用工具。

✅ 几乎适用于任何模型  
✅ 为基础或本地模型解锁工具的好方法  
❗ 链接工具时不如原生模式可靠或灵活

### 🟢 原生模式（内置函数调用）

如果您的模型支持“原生”函数调用（如 GPT-4o 或 GPT-3.5-turbo-1106），您可以使用此强大的模式让 LLM 在实时决定中调用多个工具并完成单条聊天消息中的任务。

✅ 快速、准确，一次响应可链接多个工具  
✅ 最自然和高级的体验  
❗ 需要支持原生函数调用的模型

### ✳️ 如何在模式之间切换

想在聊天中启用原生函数调用？以下是方法：

![聊天控制](/images/features/plugin/tools/chat-controls.png)

1. 打开您与模型的聊天窗口。
2. 点击 ⚙️ 聊天控制 > 高级参数。
3. 找到函数调用设置并从默认改为原生。

就这样！您的聊天现在支持真正的原生工具功能（前提是模型支持）。

➡️ 我们推荐使用 GPT-4o 或其他 OpenAI 模型以获得最佳的原生函数调用体验。  
🔎 一些本地模型可能声称支持，但在精准或复杂的工具使用上往往表现欠佳。

💡 总结：

| 模式     | 适用对象                      | 优点                                    | 缺点                                 |
|----------|----------------------------------|-----------------------------------------|--------------------------------------|
| 默认模式 | 任何模型                         | 广泛兼容、安全、灵活                   | 可能较不准确或较慢                   |
| 原生模式 | GPT-4o 等                       | 快速、智能、卓越工具链                 | 需要正确的函数调用支持               |

选择最适合您的设置——请记住，您可以随时通过聊天控制切换。

👏 到此为止 —— 您的LLM现在知道如何以及何时智能地使用工具。
