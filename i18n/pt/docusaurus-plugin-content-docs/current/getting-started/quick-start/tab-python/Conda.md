
# Instale com Conda

1. **Crie um Ambiente Conda:**

   ```bash
   conda create -n open-webui python=3.11
   ```

2. **Ative o Ambiente:**

   ```bash
   conda activate open-webui
   ```

3. **Instale o Open WebUI:**

   ```bash
   pip install open-webui
   ```

4. **Inicie o Servidor:**

   ```bash
   open-webui serve
   ```
