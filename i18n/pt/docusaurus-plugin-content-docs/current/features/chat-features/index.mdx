# Visão Geral dos Recursos de Chat

Open WebUI oferece um conjunto abrangente de recursos de chat projetados para aprimorar suas interações com os modelos de IA. Esta página fornece uma visão geral das principais capacidades de chat, com links para páginas dedicadas para informações mais detalhadas.

## Recursos Básicos de Chat

- **[🗂️ Organização de Conversas](./conversation-organization.md)**: Organize os chats com pastas e tags para manter seu espaço de trabalho limpo e estruturado.

- **[🔗 Parâmetros de URL](./url-params.md)**: Configure sessões de chat por meio de parâmetros de URL, permitindo uma configuração rápida de modelos, ferramentas e outros recursos.

- **[⚙️ Parâmetros de Chat](./chat-params.md)**: Controle os prompts do sistema e parâmetros avançados em diferentes níveis (por chat, por conta ou por modelo).

- **[🗨️ Compartilhamento de Chat](./chatshare.md)**: Compartilhe conversas localmente ou através da plataforma da Comunidade Open WebUI com configurações de privacidade controláveis.
