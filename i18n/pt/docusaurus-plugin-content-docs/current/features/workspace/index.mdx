O Espaço de Trabalho no Open WebUI fornece um ambiente abrangente para gerenciar suas interações e configurações de IA. Consiste em vários componentes principais:

## Recursos

- [🤖 Modelos](./models.md) - Crie e gerencie modelos personalizados adaptados a propósitos específicos
- [🧠 Conhecimento](./knowledge.md) - Gerencie suas bases de conhecimento para geração aumentada por recuperação
- [📚 Promptings](./prompts.md) - Crie e organize prompts reutilizáveis

Cada seção do Espaço de Trabalho é projetada para fornecer controle detalhado sobre sua experiência no Open WebUI, permitindo personalização e otimização de suas interações com IA.

## Controle de Acesso
- [‍🔑 Funções](./roles.md) - As funções definidas no Open WebUI
- [‍🔐 Grupos](./groups.md) - Configure grupos de usuários para compartilhar acesso aos recursos
- [🔒 Permissões](./permissions.md) - Configure controles de acesso e disponibilidade de recursos

Funções, grupos e permissões foram projetados para trabalhar juntos e fornecer um modelo detalhado de controle de acesso de usuário a recursos individuais.
