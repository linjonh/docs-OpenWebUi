No Open WebUI, estamos comprometidos em melhorar continuamente nossa plataforma para oferecer a melhor experiência aos nossos usuários. Abaixo, você encontrará o roteiro estruturado para nossos desenvolvimentos em andamento e futuros, categorizados em Interface, Recuperação de Informação e Comunidade.

## Interface 🖥️

Nosso roteiro para inovações na interface visa criar uma plataforma altamente intuitiva e acessível. Isso não apenas melhora a satisfação do usuário, mas também aumenta a produtividade ao reduzir a carga cognitiva dos usuários.

- 📦 **Executável Único Empacotado**: Simplificando a implantação e garantindo compatibilidade entre diferentes plataformas, pretendemos oferecer nossa interface como um executável de binário único. Essa abordagem facilitará instalação e atualizações sem esforço, melhorando significativamente a usabilidade para todos os usuários, especialmente aqueles com experiência técnica limitada.

- 👤 **Página do Usuário**: Um recurso de Página do Usuário pessoal onde os usuários podem criar postagens. A funcionalidade também incluirá recursos como seguidores, curtidas e comentários. Isso permite que os usuários compartilhem efetivamente suas configurações de modelo, prompts e arquivos com uma comunidade mais ampla, criando um ecossistema mais rico e conectado ao redor da plataforma.

- 📝 **Notas Alimentadas por IA**: Inspirado em ferramentas como Notion e Obsidian, planejamos introduzir uma funcionalidade robusta de criação de notas que inclui integração com IA. Desde anotações simples até criação de documentos completos, essa ferramenta oferecerá uma experiência contínua, tudo localmente integrado na plataforma.

- 📈 **Ferramentas Avançadas de Rastreamento de Usuário e Gerenciamento de Custos**: Os usuários terão acesso a ferramentas abrangentes projetadas para rastrear desempenho de aplicações e atividades de usuários, além de gerenciar custos de forma eficiente. Essas ferramentas capacitarão os usuários com os dados necessários para tomar decisões informadas, melhorar experiências de uso e manter controle de orçamento, otimizando o uso de recursos em suas aplicações de IA.

- 🧠 **Ferramenta de Fluxo de Trabalho com IA**: Uma ferramenta baseada em nós para orquestrar e compor múltiplos aspectos dos sistemas de IA. Essa ferramenta permitirá que os usuários conectem visualmente diferentes módulos e serviços de IA, criando fluxos de trabalho complexos com facilidade. Ela foi projetada para capacitar os usuários a aproveitar todo o potencial da IA sem necessidade de conhecimento técnico profundo em programação de IA.

- 🔊 **Integração Local de Texto para Fala**: Esse recurso permitirá o processamento de linguagem natural para converter texto em áudio falado realista, tornando a plataforma mais acessível, especialmente para usuários com deficiência visual. Essa integração ajuda a consumir informações sem interação com a tela, facilitando multitarefas e melhorando o envolvimento do usuário.

- 📣 **Detecção de Palavra-Chave de Despertar**: Incorporar operação sem uso das mãos por meio de comandos de voz não apenas aprimorará a acessibilidade, mas também alinhará com tendências emergentes em IoT e ambientes inteligentes, tornando nossa plataforma pronta para o futuro.

- 💻 **Melhor Execução de Código**: Aprimorar funcionalidades de codificação dentro da plataforma suporta uma ampla gama de atividades de desenvolvimento, tornando-a uma solução completa para desenvolvedores, atraindo uma audiência tecnológica maior e promovendo uma comunidade de desenvolvedores robusta.

- 🤖 **Função de Interpretação de Código**: Um intérprete de código flexível embutido na plataforma pode suportar múltiplas linguagens de programação, reduzindo a dependência de ferramentas externas e agilizando o processo de desenvolvimento, aumentando a eficiência.

- 🔧 **Suporte Simplificado para Ajustes Finos**: Open WebUI oferecerá um processo contínuo para ajustes finos, permitindo que os usuários construam automaticamente um conjunto de dados de ajuste fino simplesmente utilizando a interface e avaliando as respostas. Forneceremos notebooks Python para ajudar a pré-processar seu conjunto de dados, tornando-o pronto para ajustes finos diretamente do notebook. Essa abordagem será super simples e integrará elegantemente os ajustes finos aos fluxos de trabalho dos usuários.


## Recuperação de Informação (RAG) 📚

Atualmente, o Open WebUI oferece uma interface que permite o uso eficiente de modelos existentes de recuperação de informação (IR). No entanto, acreditamos que há um grande potencial para melhorias. Reconhecemos plenamente as limitações de nossa implementação atual de RAG (Geração Ampliada por Recuperação) em nosso WebUI, entendendo que uma abordagem única não atende efetivamente às necessidades de todos os usuários.

Para enfrentar esse desafio, estamos comprometidos em transformar nossa estrutura para torná-la altamente modular e extensivamente configurável. Essa nova direção permitirá que os usuários adaptem o processo de recuperação de informações às suas necessidades específicas com facilidade, utilizando uma interface interativa e simples que possibilita personalização com arrastar e soltar. Esse refinamento visa melhorar significativamente a experiência do usuário, oferecendo uma configuração IR mais adaptável e personalizada.

Aqui está nosso plano focado para avançar nossas capacidades de recuperação de informações:

- 🌐 **Framework RAG Customizável**: Visualize um futuro onde você pode personalizar facilmente sua configuração de recuperação de informações por meio de uma interface amigável. Isso incluirá componentes modulares que os usuários podem arrastar, soltar e configurar sem necessidade de conhecimento técnico profundo.

- 🔄 **Integração Avançada com Métodos SoTA**: Nosso objetivo é incorporar os últimos avanços em métodos de recuperação, melhorando a precisão e a eficiência com técnicas e arquiteturas avançadas.

- 🔍 **Pesquisa e Desenvolvimento Dedicados**: Estamos ampliando nossa pesquisa e desenvolvimento para descobrir e integrar métodos de recuperação inovadores que ultrapassam os limites dos métodos atuais, em colaboração com entidades de pesquisa líderes.

## Comunidade 🤝

Queremos criar uma comunidade vibrante onde todos possam contribuir e se beneficiar do conhecimento e dos desenvolvimentos compartilhados:

- 🏆 **Tabela de Classificação LLM**: Uma abordagem impulsionada pela comunidade para avaliar e classificar modelos garante transparência e melhoria contínua, aproveitando aplicações do mundo real para medir desempenho em vez de depender de conjuntos de dados padrão que podem não refletir a utilidade prática.

- 👥 **Subcomunidades**: Incentivar comunidades de nicho a criar conjuntos de dados e otimizar prompts pode levar a modelos melhor ajustados, já que essas comunidades têm insights específicos e feedback detalhado que normalmente passam despercebidos em conjuntos de dados mais amplos.
