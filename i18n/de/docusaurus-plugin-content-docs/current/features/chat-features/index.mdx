# Übersicht der Chat-Funktionen

Open WebUI bietet eine umfassende Reihe von Chat-Funktionen, die darauf abzielen, Ihre Interaktionen mit KI-Modellen zu verbessern. Auf dieser Seite finden Sie eine Übersicht der wichtigsten Chat-Funktionen mit Links zu detaillierten Informationsseiten.

## Kernfunktionen des Chats

- **[🗂️ Gesprächsorganisation](./conversation-organization.md)**: Organisieren Sie Chats mit Ordnern und Tags, um Ihre Arbeitsumgebung ordentlich und strukturiert zu halten.

- **[🔗 URL-Parameter](./url-params.md)**: Konfigurieren Sie Chat-Sitzungen über URL-Parameter, um Modelle, Tools und andere Funktionen schnell einzurichten.

- **[⚙️ Chat-Parameter](./chat-params.md)**: Kontrollieren Sie Systemaufforderungen und erweiterte Parameter auf verschiedenen Ebenen (pro Chat, pro Konto oder pro Modell).

- **[🗨️ Teilen von Chats](./chatshare.md)**: Teilen Sie Gespräche lokal oder über die Open WebUI-Community-Plattform mit anpassbaren Datenschutzeinstellungen.
