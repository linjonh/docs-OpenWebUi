Open WebUI のワークスペースは、AIとのやり取りや設定を管理するための包括的な環境を提供します。それはいくつかの主要なコンポーネントで構成されています：

## リソース

- [🤖 モデル](./models.md) - 特定の目的に合わせたカスタムモデルを作成および管理
- [🧠 ナレッジ](./knowledge.md) - 検索拡張生成のためのナレッジベースを管理
- [📚 プロンプト](./prompts.md) - 再利用可能なプロンプトを作成および整理

ワークスペースの各セクションは、Open WebUI の体験を微調整し、AIとのやり取りをカスタマイズおよび最適化するための詳細な制御を提供するように設計されています。

## アクセス制御
- [‍🔑 ロール](./roles.md) - Open WebUIで定義されたロール
- [‍🔐 グループ](./groups.md) - リソースへのアクセスを共有するためのユーザーグループを設定
- [🔒 権限](./permissions.md) - アクセス制御と機能の利用可能性を設定

ロール、グループ、および権限は一緒に機能して、個々のリソースへのユーザーアクセスを制御するための詳細なモデルを提供します。
