Open WebUIは、会話を管理するための強力な整理機能を提供します。ユーザーは会話を簡単に分類しタグ付けすることで、後で見つけやすく、取り出しやすくすることができます。会話を整理する主な方法は、**フォルダー**と**タグ**の2つです。

## フォルダーで会話を整理する

フォルダーを使用すると、関連する会話をまとめてグループ化し、迅速にアクセスしやすく整理することができます。

- **フォルダーを作成する**: 特定の会話を保存するための新しいフォルダーを作成できます。同じテーマや目的を持つ会話を一緒に保管したい場合に便利です。
- **会話をフォルダーに移動する**: 会話をドラッグ＆ドロップでフォルダーに移動できます。これにより、自分の作業スタイルに適した形でワークスペースを構築できます。

![Folder Demo](/images/folder-demo.gif)

### 利用例

:::tip **プロジェクトごとに整理**
複数のプロジェクトを管理している場合は、それぞれのプロジェクト用に個別のフォルダーを作成し、関連する会話をこれらのフォルダーに移動できます。これにより、プロジェクト関連のすべてのディスカッションを1か所にまとめることができます。
:::

## 会話のタグ付け

タグを使用すると、キーワードやフレーズで会話をラベル付けすることで、さらに詳細な整理が可能になります。

- **会話にタグを追加する**: 会話の内容や目的に応じてタグを適用できます。タグは柔軟に追加・削除できるため、必要に応じて変更が可能です。
![Tag Demo](/images/tag-demo.gif)
- **検索にタグを活用する**: タグを使用すると、検索機能を使って特定の会話を簡単に見つけることができます。タグでフィルタリングすることで、特定のトピックに関連する会話をすばやく見つけることができます。

### 利用例

:::tip **トピックごとにタグ付け**
「マーケティング」や「開発」など、特定のトピックについて頻繁に議論する場合、これらの用語で会話をタグ付けすることができます。後で特定のタグを検索すれば、関連するすべての会話に迅速にアクセスできます。
:::
