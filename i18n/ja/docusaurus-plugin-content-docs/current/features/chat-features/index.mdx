# チャット機能の概要

Open WebUIは、AIモデルとの対話を向上させるための包括的なチャット機能セットを提供します。このページでは、主要なチャット機能の概要と、詳細情報を記載した専用ページへのリンクを紹介します。

## 核心チャット機能

- **[🗂️ 会話の整理](./conversation-organization.md)**: フォルダやタグを使用してチャットを整理し、作業環境を整え構造化します。

- **[🔗 URLパラメータ](./url-params.md)**: URLパラメータを使用してチャットセッションを設定し、モデルやツールなどの機能を迅速に構成します。

- **[⚙️ チャットパラメータ](./chat-params.md)**: システムプロンプトや高度なパラメータを異なるレベル（チャットごと、アカウントごと、モデルごと）で制御します。

- **[🗨️ チャット共有](./chatshare.md)**: ローカルまたはOpen WebUIコミュニティプラットフォームを通じて、プライバシー設定を管理しながら会話を共有します。
