## 概要

Open WebUIは、OpenAIやその他のOpenAI互換APIに簡単に接続して使用できるようにするツールです。このガイドでは、APIキーの追加、正しいエンドポイントの設定、モデルの選択について説明し、すぐにチャットを始められるようになる手順を案内します。

---

## ステップ1: OpenAI APIキーを取得する

OpenAIモデル（GPT-4やo3-miniなど）を使用するためには、対応プロバイダーからAPIキーを取得する必要があります。

利用可能な選択肢:

- OpenAI公式 (https://platform.openai.com/account/api-keys)
- Azure OpenAI
- その他のOpenAI互換サービス (例: LocalAI, FastChat, Helicone, LiteLLM, OpenRouterなど)

👉 キーを取得したら、コピーしてすぐに使えるように準備してください。

ほとんどのOpenAI利用では、デフォルトのAPI基本URLは以下の通りです:  
https://api.openai.com/v1

他のプロバイダーは異なるURLを使用します—プロバイダーのドキュメントを確認してください。

---

## ステップ2: Open WebUIでAPI接続を追加

Open WebUIが稼働している場合:

1. ⚙️ **管理設定**に移動します。
2. **接続 > OpenAI > 管理**に移動します（レンチアイコンを探してください）。
3. ➕ **新しい接続を追加**をクリックします。
4. 以下の情報を入力してください:
   - API URL: https://api.openai.com/v1（または特定のプロバイダーのURL）
   - APIキー: ここにキーを貼り付けてください

5. ✅ 保存をクリックします。

これで認証情報が安全に保存され、接続が設定されます。

設定画面は以下のようになります:

![OpenAI接続画面](/images/getting-started/quick-start/manage-openai.png)

---

## ステップ3: モデルの使用を開始

接続が保存されたら、Open WebUI内でモデルを使用できます。

🧠 モデルをダウンロードする必要はありません—モデルセレクターから1つを選択してチャットを開始するだけです。プロバイダーがサポートしているモデルであれば、そのAPIを通じてすぐに使用可能です。

モデル選択画面は以下のようになります:

![OpenAIモデルセレクター](/images/getting-started/quick-start/selector-openai.png)

GPT-4、o3-mini、またはプロバイダーが提供する互換モデルを選択してください。

---

## 準備完了!

これで完了です！OpenAI互換API接続が利用可能になりました。

Open WebUIとOpenAIを使えば、強力な言語モデル、直感的なインターフェース、即時チャット機能へのアクセスが可能になります—設定の手間を省いて利用できます。

問題が発生した場合や追加サポートが必要な場合は、[ヘルプセクション](/troubleshooting)をご覧ください。
