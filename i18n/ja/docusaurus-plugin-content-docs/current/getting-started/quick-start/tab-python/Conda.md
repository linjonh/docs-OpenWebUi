
# Condaでインストール

1. **Conda環境を作成する:**

   ```bash
   conda create -n open-webui python=3.11
   ```

2. **環境をアクティベートする:**

   ```bash
   conda activate open-webui
   ```

3. **Open WebUIをインストールする:**

   ```bash
   pip install open-webui
   ```

4. **サーバーを起動する:**

   ```bash
   open-webui serve
   ```
