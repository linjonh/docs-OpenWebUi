
# Установка с помощью Conda

1. **Создайте окружение Conda:**

   ```bash
   conda create -n open-webui python=3.11
   ```

2. **Активируйте окружение:**

   ```bash
   conda activate open-webui
   ```

3. **Установите Open WebUI:**

   ```bash
   pip install open-webui
   ```

4. **Запустите сервер:**

   ```bash
   open-webui serve
   ```
