# Список переменных окружения


:::info
Для полного списка всех переменных окружения Open WebUI см. страницу [Конфигурация переменных окружения](/getting-started/env-configuration).
:::

Ниже приведено краткое описание переменных окружения для преобразования речи в текст (STT).

# Переменные окружения для преобразования речи в текст (STT)

| Переменная | Описание |
|----------|-------------|
| `WHISPER_MODEL` | Устанавливает модель Whisper для локального преобразования речи в текст |
| `WHISPER_MODEL_DIR` | Указывает директорию для хранения файлов модели Whisper |
| `WHISPER_LANGUAGE` | Указывает язык преобразования речи в текст для Whisper в формате ISO 639-1 (ISO 639-2 для гавайского и кантонского), если язык не указан, он предсказывается |
| `AUDIO_STT_ENGINE` | Указывает движок для преобразования речи в текст (пусто для локального Whisper или `openai`) |
| `AUDIO_STT_MODEL` | Указывает модель преобразования речи в текст для совместимых с OpenAI конечных точек |
| `AUDIO_STT_OPENAI_API_BASE_URL` | Устанавливает базовый URL-адрес для преобразования речи в текст, совместимый с OpenAI |
| `AUDIO_STT_OPENAI_API_KEY` | Устанавливает API-ключ OpenAI для преобразования речи в текст |
