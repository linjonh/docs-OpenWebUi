:::info **關於用戶角色和隱私的重要提示：**

- **管理員建立：** 在 Open WebUI 上創建的第一個帳戶將獲得**管理員權限**，可以控制用戶管理和系統設置。
- **用戶註冊：** 之後的註冊用戶默認處於**待定**狀態，需要管理員批准才能訪問。
- **隱私與數據安全：** **所有數據**（包括登錄資料）都將**本地存儲**在您的設備上。Open WebUI 確保**嚴格保密**並**無外部請求**，以增強隱私和安全性。
  - **所有模型默認為私有。** 模型必須通過群組顯式共享或設為公開。如果將模型分配給群組，只有該群組的成員可以看到它。如果將模型設置為公開，則實例中的任何人都可以看到。

:::

請在下面選擇您偏好的安裝方法：

- **Docker：** **官方支持，並建議大多數用戶使用**
- **Python：** 適合資源有限的環境或需要手動設置的用戶
- **Kubernetes：** 適合需要擴展和編排的企業部署

## 下一步

安裝完成後，訪問：

- [http://localhost:3000](http://localhost:3000) 獲取 Open WebUI 的訪問地址。
- 或 [http://localhost:8080/](http://localhost:8080/) 當使用 Python 部署時。

您現在可以開始使用 Open WebUI！
