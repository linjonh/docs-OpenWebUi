Open WebUI 的工作區提供了一個全面的環境，用於管理您的人工智慧互動和配置。它包含幾個關鍵組件：

## 資源

- [🤖 模型](./models.md) - 創建和管理針對特定用途定制的模型
- [🧠 知識](./knowledge.md) - 管理您的知識庫以進行檢索增強生成
- [📚 提示](./prompts.md) - 創建和組織可重用的提示

工作區的每個部分都旨在為您提供對 Open WebUI 體驗的細緻控制，允許定制和優化您的人工智慧互動。

## 訪問控制
- [‍🔑 角色](./roles.md) - Open WebUI 中定義的角色
- [‍🔐 群組](./groups.md) - 設置用戶群組以共享對資源的訪問
- [🔒 權限](./permissions.md) - 配置訪問控制和功能可用性

角色、群組和權限被設計為一起工作，以提供一個細緻的模型來控制用戶對個別資源的訪問。
