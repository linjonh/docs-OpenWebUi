# 聊天功能概述

Open WebUI 提供一套全面的聊天功能，旨在提升您與 AI 模型的互動體驗。本頁面提供關鍵聊天功能的概述，以及指向詳細信息的專用頁面。

## 核心聊天功能

- **[🗂️ 聊天組織](./conversation-organization.md)**：通過文件夾和標籤組織聊天，保持您的工作區整潔和結構化。

- **[🔗 URL 參數](./url-params.md)**：通過 URL 參數配置聊天會話，快速設置模型、工具和其他功能。

- **[⚙️ 聊天參數](./chat-params.md)**：在不同層級（每個聊天、每個帳戶或每個模型）控制系統提示和高級參數。

- **[🗨️ 聊天分享](./chatshare.md)**：通過本地或 Open WebUI 社區平台分享聊天，並控制隱私設定。
