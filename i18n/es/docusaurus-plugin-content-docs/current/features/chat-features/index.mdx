# Resumen de Características del Chat

Open WebUI ofrece un conjunto completo de características de chat diseñadas para mejorar tus interacciones con modelos de IA. Esta página proporciona una descripción general de las capacidades clave del chat, con enlaces a páginas dedicadas para obtener información más detallada.

## Características Principales del Chat

- **[🗂️ Organización de Conversaciones](./conversation-organization.md)**: Organiza chats con carpetas y etiquetas para mantener tu espacio de trabajo limpio y estructurado.

- **[🔗 Parámetros de URL](./url-params.md)**: Configura sesiones de chat a través de parámetros de URL, lo que permite una configuración rápida de modelos, herramientas y otras características.

- **[⚙️ Parámetros de Chat](./chat-params.md)**: Controla los mensajes del sistema y parámetros avanzados a diferentes niveles (por chat, por cuenta o por modelo).

- **[🗨️ Compartir Chats](./chatshare.md)**: Comparte conversaciones localmente o a través de la plataforma de la Comunidad Open WebUI con configuraciones de privacidad controlables.
