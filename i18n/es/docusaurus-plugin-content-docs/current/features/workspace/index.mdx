El Espacio de Trabajo en Open WebUI proporciona un entorno integral para gestionar tus interacciones y configuraciones de IA. Consta de varios componentes clave:

## Recursos

- [🤖 Modelos](./models.md) - Crear y gestionar modelos personalizados adaptados a propósitos específicos
- [🧠 Conocimiento](./knowledge.md) - Gestiona tus bases de conocimiento para la generación aumentada por recuperación
- [📚 Indicaciones](./prompts.md) - Crear y organizar indicaciones reutilizables

Cada sección del Espacio de Trabajo está diseñada para ofrecerte un control detallado sobre tu experiencia en Open WebUI, permitiendo personalización y optimización de tus interacciones de IA.

## Control de Acceso
- [‍🔑 Roles](./roles.md) - Los roles definidos en Open WebUI
- [‍🔐 Grupos](./groups.md) - Configurar grupos de usuarios para compartir acceso a recursos
- [🔒 Permisos](./permissions.md) - Configurar controles de acceso y disponibilidad de funciones

Roles, grupos y permisos están diseñados para trabajar juntos y proporcionar un modelo detallado para controlar el acceso de los usuarios a recursos individuales.
