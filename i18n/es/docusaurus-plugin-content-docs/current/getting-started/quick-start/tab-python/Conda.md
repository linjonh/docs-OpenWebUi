
# Instalar con Conda

1. **Crear un entorno Conda:**

   ```bash
   conda create -n open-webui python=3.11
   ```

2. **Activar el entorno:**

   ```bash
   conda activate open-webui
   ```

3. **Instalar Open WebUI:**

   ```bash
   pip install open-webui
   ```

4. **Iniciar el servidor:**

   ```bash
   open-webui serve
   ```
